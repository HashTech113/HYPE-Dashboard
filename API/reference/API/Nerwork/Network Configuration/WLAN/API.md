# WLANScan

This API is used for get or set WIFI parameters.

## URI

POST /API/NetworkConfig/ScanWlan/{Action1}
POST /API/NetworkConfig/MacthWiFiType/{Action2}
POST /API/APNetworkCfg/{Action3}
